# Регламент поступления

## Условия поступления

1. Возраст: с 14 лет.
2. Оконченная музыкальная школа по классу фортепиано.
3. Письмо от церкви.

## Вступительный экзамен

### Для вновь поступающих

1. **Служение Словом**

   Необходимо выучить все билеты.

2. **Теория музыки и сольфеджио**

   Необходимо подготовить все билеты.

3. **Специальность**

   Необходимо подготовить:

   - гамму;
   - сольное произведение наизусть;
   - чтение с листа;
   - аккомпанемент по слуху.

#### Требования к сольному произведению

Сольное произведение исполняется наизусть. Необходимо сыграть обработку для фортепиано соло.

Примеры произведений можно посмотреть в сборниках:

- «Нотная тетрадь пианиста»;
- «Klavierheft».

#### Требования к чтению с листа

Для чтения с листа используются хоровые партитуры из 1-го и 2-го тома сборника «Песни христиан».

### Для продолжающих обучение

Необходимо подготовить только сольное произведение наизусть: обработку для фортепиано соло.
